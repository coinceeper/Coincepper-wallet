# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability, **do NOT file a public issue**. Report it privately to **security@coinceeper.com**.

### Response Timeline

- **Acknowledgment**: Within 48 hours
- **Verification**: Within 5 business days
- **Fix**: Typically 1-14 days depending on severity

## Supported Versions

| Version | Supported |
|---------|-----------|
| 1.0.x   | ✅ Active |

## Security Features

- Non-custodial architecture: keys never leave the device
- HD Wallet: BIP39 + BIP32/BIP44/BIP84 standards
- Secure storage: Android Keystore / iOS Keychain
- Encryption: AES-256-GCM with PBKDF2 (120K iterations)
- Biometric authentication: Face ID / fingerprint
- TLS pinning for API communications
